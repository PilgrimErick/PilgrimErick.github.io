今天群里聊到星星评分实现的需求，突然想起来以前看过此类的一个超简单实现，花了几分钟思考重现了这一小段代码：

```Javascript
    /*
     *  score为评分，范围为0-5 
     */
    function stars(score){
      return "★★★★★☆☆☆☆☆".substr(5-score, 5);
    }

    stars(3);   //★★★☆☆
```
